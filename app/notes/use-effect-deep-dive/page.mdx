# 深入理解 useEffect（React Hook）

你是否曾經寫過像這樣的程式碼？

```tsx
useEffect(() => {
  console.log("Component mounted");
}, []);
```

### 📌 常見錯誤

- 忘記加依賴
- 加了 setState 導致無限 re-render
- 不理解 cleanup 的必要性

### ✅ 正確用法示範

```tsx
useEffect(() => {
  const id = setInterval(() => {
    console.log("interval tick");
  }, 1000);

  return () => clearInterval(id); // 清理資源
}, []);
```
